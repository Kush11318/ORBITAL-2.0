3. METHODS & MATERIALS
--------------------------------------------------------------------------------
* The 3D graphics environment is built using vanilla JavaScript, Three.js, and GLTFLoader to import high-fidelity 3D models of the Earth and Moon.
* Atmospheric radiance and glowing sci-fi hologram visual overlays are achieved using the Three.js EffectComposer with UnrealBloomPass post-processing passes.
* Data telemetry is retrieved directly from NASA and JPL APIs, including planetary APOD, Close Approach Data (CAD), NeoWs, Sentry impact monitoring, and DONKI space weather reports.
* Real-time satellite positions and orbital parameters are fetched from CelesTrak's GP JSON API to render accurate orbital element catalogs.
* To prevent IP-based rate limiting and overcome CORS limitations, a robust array-based proxy fallback mechanism (AllOrigins, CORSProxy, CodeTabs) is integrated into all API services.
* Dynamic UI features include customized CSS grid panels, interactive buttons, coordinate mapping displays, and scrolling trackers synced to scroll events.
* Procedural asset generation is utilized, such as programmatically drawing radial gradients onto canvas elements to build nebula clouds without loading heavy image assets.
